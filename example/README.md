# Example —— 文件整理指南参考骨架

本目录是 [personal_file_guide_zh.md](../personal_file_guide_zh.md) 中定义的目录结构的示例实现，可直接复制到 `D:\` 作为初始骨架。

## 使用方式

1. 将本目录下的 10 个顶层文件夹复制到 `D:\`
2. 去掉 `example\` 这一层，使路径变为 `D:\Archive\`、`D:\Courses\` 等
3. 开始使用——后续文件归位有了清晰的目标位置

## 已实现的目录

- `Archive\` — 归档旧项目、历史资料（按年分）
- `Courses\` — 课程学习（按学期 → 课程）
- `Dev\` — 通用开发、脚本、练习
- `Docs\` — 个人资料、证书、日记
- `Downloads\` — 临时下载中转
- `Installers\` — 安装包、镜像、驱动
- `Lab\` — 实验室事务
- `Media\` — 图片、视频、音频、字体、截图
- `Research\` — 科研主线
- `Systems\` — 软件运行环境
